Quem Somos

O Universo em Foco é um projeto educacional criado com o objetivo de aproximar estudantes e entusiastas das descobertas científicas que ajudam a expandir o conhecimento humano sobre o espaço.

Nosso trabalho consiste em reunir informações sobre missões espaciais, telescópios, observatórios, planetas e fenômenos astronômicos, apresentando conteúdos de forma clara e acessível para todos os públicos.

Acreditamos que a ciência é uma ferramenta fundamental para o desenvolvimento da sociedade e que a divulgação científica pode inspirar novas gerações a seguirem carreiras ligadas à tecnologia, pesquisa e inovação.

Este projeto foi desenvolvido como atividade acadêmica, utilizando tecnologias web como HTML, CSS e JavaScript para criar uma experiência informativa e interativa para os visitantes.

Nossa missão é despertar a curiosidade sobre o universo e mostrar como grandes projetos científicos, como o Telescópio Espacial James Webb, contribuem para ampliar os limites do conhecimento humano.
